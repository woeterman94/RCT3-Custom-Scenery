EpcotPath
=========

A sci-fi style custom path based on the Epcot bridge.

Models and textures by Old-Spice.
Path work by Jdrowlands.

EpcotPath built on 16.05.10.
Path created with SketchUp7.1, Importer18 and ovlmake0.5.

Install
-------

To install, copy the contents of 'RCT3 Installation Directory' to your RCT3 Installation Directory. Do *not* extract to Style\Themed.

Make sure your unpacker preserves the directory structure! If you are asked if you want to merge, say yes.

Uninstall
---------

Remove the 'Epcot' directory from RCT3dir\Path\.
